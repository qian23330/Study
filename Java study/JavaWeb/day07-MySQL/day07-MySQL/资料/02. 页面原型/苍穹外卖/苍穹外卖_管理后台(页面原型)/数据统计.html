1、选择时间器：默认为近7日，可以选择 昨日、近7日、近30日、本周、本月

1.1 近7日：今日往前推7个自然日

1.2 近30日：今日往前推30个自然日

1.3 本周：按自然周统计数据，统计数据为本周一至今日

1.4 本月：按自然月统计数据，统计数据为本月1日至今日


2、数据概览展示统计数据结果，包括数据统计值及同比数据。

2.1 营业额：根据时间选择区间，汇总已完成订单金额。

2.2 有效订单：根据订单状态统计有效订单数，汇总已完成订单数量。

2.3 订单完成率=有效订单数/总订单数*100%

2.4 平均客单价=营业额/有效订单数

2.5 用户总量：本店总体用户量

2.6 新增用户：根据时间选择区间，统计新增用户数量


3、营业额统计根据时间选择区间，展示营业额与订单数的趋势数据

3.1 左侧为营业额数据坐标轴

3.2 右侧为订单数数据坐标轴


4、用户统计根据时间选择区间，展示用户总量和新增用户的趋势数据

4.1 左侧为用户总量数据坐标轴

4.2 右侧为新增用户数据坐标轴


5、销量排名根据时间选择区间，展示销量前10的菜品信息
