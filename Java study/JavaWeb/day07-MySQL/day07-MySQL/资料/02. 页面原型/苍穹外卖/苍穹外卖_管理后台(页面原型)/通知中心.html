消息通知：以通知的形式，提醒商家应该注意的关键信息，并完成相关的操作

1. 点击消息通知按钮，进入通知中心页面；

2. 未读状态下，点击全部已读按钮，当前页信息已读

3. 已读状态下，全部已读按钮为不可点击状态；

4. 未读、已读按照时间倒序排列

5. 未读仅展示时间最近的200条信息，已读仅展示最近的300条信息；

6. 每页默认展示10条通知；


通用操作说明：

1. 待接单、待派送通知，需点击跳转后，该条信息标记已读；若商户在没有查看通知的情况下，在订单管理中已经处理了订单，通知界面依然是未读，

2. 催单和闭店，鼠标悬停在标题上，下方展示完整的信息，当鼠标悬停超过2秒，未读红点消失，

当刷新页面，或跳转到其他页面时，该条信息显示在已读列表中；
